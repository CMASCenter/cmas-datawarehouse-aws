2018_12US1 CMAQ Modeling Platform Data

Description of the data.
This bucket contains input data to run 2018 12US1 CMAQ for two different emissions scenariors. This data uses netCDF4 compression.
For a complete description with metadata please see the following dataverse entries:
CMAQ Model Version 5.4 CB6r5 Input Data 1/1/2018-12/31/2018
and
CMAQ Model Version 5.4 CRACMM Input Data -- 1/1/2018 - 12/31/2018 12km CONUS
Example listing of the directory and emission file names:
cb6r3_ae6_20200131_MYR: emis_mole_rwc_20171220_12US1_cmaq_cb6_WR413_MYR_2017.nc4 
WR705_2018gc2_cracmmv1: emis_mole_rwc_20171222_12US1_cmaq_cracmmv1_WR705_2018gc2.ncf

Downloading data from S3 Bucket

This dataset is available as part of the AWS Open Data Program, therefore egress fees are not charged to either the host or the person downloading the data.
Instructions for downloading the aws command line.

Instructions to Download AWS Command Line
Once you have it installed, you can use it without credentials.
Example commands:
To query how much data is in a bucket: 
aws s3 ls --summarize --region=us-east-1 --human-readable --recursive s3://cmas-cmaq-modeling-platform-2018
Output: 
Total Objects: 18579
   Total Size: 6.1 TiB
Command to copy files from the bucket to your local filesystem (the system where you are running the aws cli) and place it under the path that you specify for /your_local_path. 
aws s3 --no-sign-request --region=us-east-1 cp --recursive s3://cmas-cmaq-modeling-platform-2018 /your_local_path/
To obtain just the files you need from the CMAQ input data and avoid downloading the full bucket you can use the --include and --exclude commands and use the --dryrun option to see what would be copied before you begin.
AWS CLI S3 Copy command - latest reference manual 
aws s3 ls --summarize --region=us-east-1 --human-readable --recursive s3://cmas-cmaq-modeling-platform-2018/2018_12US1
Example, to only copy the README to your local directory "." using use (remove the --dryrun option): 
aws s3 cp --dryrun --region=us-east-1 --exclude "*" --include "CMAQ_Control_DESID.nml" --recursive s3://cmas-cmaq-modeling-platform-2018/CMAQ_12US1 .
You can also use the aws s3 cp command to obtain specific dates using the above method.
