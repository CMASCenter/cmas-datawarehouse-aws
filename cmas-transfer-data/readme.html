Instructions to Download AWS Command Line
Once you have it installed, you can use it without credentials.
Example commands:

To query how much data is in the full bucket:


aws s3 ls --no-sign-request --summarize --region=us-east-1 --human-readable --region=us-east-1 --recursive s3://cmas-transfer-data
The above command will take some time, as there are approximately 2.37 TB of data on the bucket.

To copy or download all of this data to your local directory , use the following command:

aws s3 --no-sign-request --region=us-east-1 cp --recursive s3://cmas-data-transfer /your/local/directory/
