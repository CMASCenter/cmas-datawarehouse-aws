2018v2 Emissions Modeling Platform

This s3 bucket contains emissions for 2018v2 platform cases.
This platform was used to support the development of the National Ambient Air Quality Standards for Particulate Matter (PM NAAQS) issued by the U.S. EPA on February 7, 2024
National Ambient Air Quality Standards (NAAQS) for Particulate Matter
The BEIS and fertilizer were not used as inputs to the CMAQ model but are provided as these inputs are required for CAMx.

Instructions for downloading the aws command line.


Instructions to Download AWS Command Line
Once you have it installed, you can use it without credentials.
Example commands:

To query how much data is in the full bucket:


aws s3 ls --no-sign-request --summarize --region=us-east-1 --human-readable --region=us-east-1 --recursive s3://2018v2platform
Tail of Output:

   Total Size: 3.9 TiB

To copy or download all of this data, if you have 3.9 TiB of local disk space, use the following command:



aws s3 --no-sign-request --region=us-east-1 cp --recursive s3://2018v2platform /your/local/directory/



A script is being developed for downloading 2 days of input data to run a benchmark using this case. That script can be modified to obtain files required to model for a user specified time period.
