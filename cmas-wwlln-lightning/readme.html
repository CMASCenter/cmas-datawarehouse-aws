Gridded Hourly WWLLN Lightning Flash Density Data -- 1/1/2016 - 12/31/2023 12km CONUS

Description of the data.
CMAQ-ready gridded hourly observed lightning flash density data for the 12US1 domain from the World Wide Lightning Location Network (WWLLN) with scaling (WWLLNs). These data were processed by Daiwen Kang of EPA Office and Research Development and were updated on 11/19/2024. Metadata information available on Dataverse for CMAS Data Warehouse Gridded Hourly WWLLN Lightning Flash Density Data -- 1/1/2016 - 12/31/2022 12km CONUS

Downloading data from S3 Bucket

This dataset is available as part of the AWS Open Data Program, therefore egress fees are not charged to either the host or the person downloading the data.
Instructions for downloading the aws command line.

Instructions to Download AWS Command Line

aws s3 ls --summarize --region=us-east-1 --human-readable --recursive s3://cmas-wwlln-lightning
