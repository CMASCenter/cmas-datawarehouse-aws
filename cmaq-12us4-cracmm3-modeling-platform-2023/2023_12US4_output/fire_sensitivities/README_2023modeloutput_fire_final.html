DESCRIPTION
This dataset includes CMAQ CRACMM output for Summer 2023 across the U.S. and Canada for a base simulation and simulation without fires.

CMAQ INPUTS
The CMAQ inputs, including meteorology, emissions, boundary conditions, initial conditions, and additional required inputs (US EPA, 2025) used in creating these 2023 output data are available at https://doi.org/10.15139/S3/GL41QC. A CMAQ run script for these inputs is available from Pye (2025). CMAQ is available from the US EPA Office of Research and Development (2024) and at https://github.com/USEPA/CMAQ. CRACMM is available from the US EPA Office of Research and Development (2025) and at https://github.com/USEPA/CRACMM.

CMAQ SIMULATIONS
These output data are from CMAQ simulations for 2023 across the U.S. and Canada. Data in the base simulation output was extracted from a full 2023 annual simulation. The base simulation largely followed the platform used by Pye et al. (2026) and scripts available from Pye (2025) except this work uses CMAQv5.5 with additional updates including standard CRACMM version 3 (US EPA Office of Research and Development, 2025). The base simulation has a paired “nofire” simulation in which all fire emissions (agricultural, outside U.S., prescribed, and U.S. wildland fires) were turned off. The nofire simulation started April 2, 2023, and April data was discarded as initialization spinup. These files contain hourly or daily values of pollutants. The user is encouraged to check for large values during fire events that may influence their analysis.

CMAQ HOURLY GRIDDED SURFACE DATA FOR JUNE 2023
The two netcdf COMBINE*202306.nc files (for base and nofire) contain hourly average, surface level predictions for the month of June 2023 with horizontal dimensions of 354 rows and 459 columns. The species in these files are defined by SpecDef_Conc_cracmm3.txt as a combination of base CRACMM species (see work of US EPA Office of Research and Development, 2025 and https://github.com/USEPA/CRACMM for species metadata). All time steps in these files are in GMT.

CMAQ HOURLY GRIDDED 3D DATA FOR JUNE 2023
The 60 daily netcdf files CCTM_CONC*202306*.nc have a vertical resolution of 35 layers for 354 rows and 459 columns and contain instantaneous, hourly model predictions for a select number of species: carbon monoxide (CO), ozone (O3), nitric oxide (NO), nitrogen dioxide (NO2), acetaldehyde (ACD), formaldehyde (HCHO), peroxyacetyl nitrate and similar species (PAN), benzene (BEN), toluene (TOL), nitric acid (HNO3), dinitrogen pentoxide (N2O5), lumped anthropogenic organic nitrates (ONIT), Aitken mode nitrate aerosol (ANO3I), accumulation mode aerosol nitrate (ANO3J), coarse mode aerosol nitrate (ANO3K), vertical velocity component (W_VEL), air temperature (TA), and pressure (PRES). These files are provided for base and nofire simulations for all days in June. All time steps in these files are in GMT.

CMAQ DAILY AVERAGE GRIDDED SURFACE DATA FOR MAY – SEPTEMBER 2023
The two HR2DAY*.nc files are surface level model predictions (354 rows by 459 columns) for maximum daily 8-hr average ozone (O3_M3A8), daily average ozone (O3_DAILYAV), 1 hour daily maximum ozone (O3_MAX), daily average fine particulate matter of 2.5 micrometers in diameter or less (PM25_TOT_AVG), daily average sulfate in fine particulate matter (PM25_SO4_AVG), daily average nitrate in fine particulate matter (PM25_NO3_AVG), daily average ammonium in fine particulate matter (PM25_NH4_AVG), daily average organic carbon in fine particulate matter (PM25_OC_AVG), and daily average elemental carbon in fine particulate matter (PM25_EC_AVG). These files contain information for May 1 to September 30, 2023 and all daily metrics were calculated using the 24 hours of hourly average model output corresponding to hours 0 – 23 of local standard time at a given grid cell. These files are provided for base and nofire simulations for days in June.

CMAQ HOURLY VERTICAL PROFILES AT SELECT LOCATIONS
The daily EXTRACT*csv files (within tar files) contain vertical profiles of hourly values of CMAQ full layer height above ground level (ZF), CMAQ layer pressure (PRES), O3, PAN, ACD, and CO for select locations in Wisconsin, Connecticut, New Jersey, and/or Indiana (locations indicated in files). The Jupyter notebook (ipynb file) used to generate these files is also provided. These files are for the base simulation only. All times in these files are in GMT.

FILE FORMAT
The 2023 model outputs in .nc files are stored as uncompressed netcdf formatted files using I/O API data structures. Information on the model projection and grid structure as well as variable names and units are contained in the header information of the netcdf files. The netcdf files can be opened and manipulated using I/O API utilities (e.g. M3XTRACT, M3WNDW) or other software programs that can read and write netcdf formatted files (e.g. Fortran, R, Python). The Species Definition file is a plain text file. The vertical profiles at select locations are comma separated (csv) files. An example of how to read the csv files is provided in the ipynb jupyter notebooks.
