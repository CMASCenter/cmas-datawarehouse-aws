Instructions for downloading the aws command line.

Instructions to Download AWS Command Line
Once you have it installed, you can use it without credentials.
Example commands:
To list the contents of the CMAQv5.4 2018 12NE3 Benchmark 2Day Input Bucket 
aws s3 --no-sign-request ls --summarize --region=us-east-1 --human-readable --recursive s3://cmas-cmaq/CMAQv5.4_2018_12NE3_Benchmark_2Day_Input

To recursively copy this data to a local directory use the command
aws s3 --no-sign-request cp --region=us-east-1 --recursive s3://cmas-cmaq/CMAQv5.4_2018_12NE3_Benchmark_2Day_Input .
