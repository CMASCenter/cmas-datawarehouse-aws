Instructions to Download AWS Command Line
Once you have it installed, you can use it without credentials.
Example commands:
To query how much data is in a bucket: 
aws s3 ls --summarize --region=us-east-1 --human-readable --recursive s3://2016v3platform

To copy or download the data:
aws s3 --no-sign-request --region=us-east-1 cp --recursive s3://2016v3platform /your/local/directory
To list and/or download a subset of the data use the --include and --exclude options use the following command (remove the --dryrun option after you have confirmed that the files exist.) 
aws s3 --no-sign-request cp --dryrun --region=us-east-1 --exclude "*" --include "*METDOT3D*" --recursive s3://mpas-cmaq .
